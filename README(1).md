# Transport & Bus Ticketing System - Oracle PL/SQL Capstone

## Phase I: Problem Statement

This project focuses on designing a production-ready Oracle database for
a Transport & Bus Ticketing System with full Business Intelligence
capabilities.

## Features

-   Ticket booking & seat allocation
-   Route & schedule management
-   Payment tracking & reconciliation
-   BI dashboards and analytics

## Project Phases

1.  Problem Identification
2.  Business Process Modeling
3.  ER Design & Data Dictionary
4.  Oracle PDB Creation
5.  Table Implementation
6.  PL/SQL Development
7.  Triggers, Security & Auditing
8.  BI Integration & Final Presentation
